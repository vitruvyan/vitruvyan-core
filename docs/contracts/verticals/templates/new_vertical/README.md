# New Vertical Template

Use this template to bootstrap a vertical compliant with Vertical Contract V1.

## Required files

- `intent_config.py`
- `README.md`
- `vertical_manifest.yaml`
- `tests/test_vertical_contract.py`

## Recommended files

- `graph_plugin.py`
- `governance_rules.py`
- `slot_filler.py`
- `response_formatter.py`

## Validation workflow

1. Fill `vertical_manifest.yaml`
2. Implement required components
3. Run contract tests
4. Run integration tests
5. Complete checklist in `docs/contracts/verticals/VERTICAL_CONFORMANCE_CHECKLIST.md`
